# Engenharia de Requisitos - Eletrostart

## 1. Visão Geral
Desenvolvimento de um website institucional para a empresa Eletrostart, especializada em materiais elétricos (casa, empresas, indústrias) e placas solares. O objetivo é apresentar a empresa, seus produtos/serviços e facilitar o contato com clientes.

## 2. Público-Alvo
- Proprietários de residências.
- Gestores de empresas e indústrias.
- Pessoas interessadas em energia solar.
- Eletricistas e profissionais da área.

## 3. Requisitos Funcionais

### RF01 - Página Inicial (Home)
- Deve conter um banner (Hero Section) com a proposta de valor e imagens de alta qualidade.
- Deve apresentar os principais serviços/produtos.
- Deve conter chamadas para ação (CTAs) para contato ou orçamento.

### RF02 - Seção de Serviços/Produtos
- Listagem das áreas de atuação: Materiais Elétricos (Residencial, Comercial, Industrial) e Energia Solar.
- Descrição breve de cada serviço.

### RF03 - Seção Sobre Nós
- História da empresa, missão, visão e valores.
- Destaque para a experiência e qualidade.

### RF04 - Seção de Contato
- Formulário de contato (Nome, E-mail, Telefone, Mensagem).
- Informações de endereço, telefone e e-mail.
- Mapa de localização (opcional).
- Links para redes sociais.

### RF05 - Responsividade
- O site deve se adaptar a dispositivos móveis (celulares e tablets) e desktops.

## 4. Requisitos Não Funcionais

### RNF01 - Desempenho
- O site deve carregar rapidamente (boas práticas de performance, imagens otimizadas).

### RNF02 - Usabilidade
- Navegação intuitiva e fácil.
- Design limpo e profissional, seguindo a identidade visual (Azul, Vermelho, Branco).

### RNF03 - SEO (Search Engine Optimization)
- Estrutura de código semântica (HTML5).
- Meta tags configuradas.

### RNF04 - Tecnologia
- Frontend: React.js (Vite).
- Estilização: Tailwind CSS.
- Ícones: Lucide React.
